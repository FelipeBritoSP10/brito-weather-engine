# 🌦️ Brito Weather Engine

Uma plataforma climática moderna desenvolvida com React, TypeScript e Mantine UI, focada em experiência visual, arquitetura escalável e performance.

---

# 🚀 Sobre o Projeto

O **Brito Weather Engine** é um dashboard climático moderno criado para oferecer uma experiência intuitiva e elegante para visualização de informações meteorológicas.

O projeto foi desenvolvido utilizando tecnologias modernas do ecossistema React e possui uma estrutura preparada para crescimento e futuras integrações.

# ⚙️ Instalação

Clone o projeto:

```bash
git clone https://github.com/FelipeBritoSP10/brito-weather-engine
```

Entre na pasta do projeto:

```bash
cd brito-weather-engine
```

Instale as dependências:

```bash
npm install
```

---

# ▶️ Executando o Projeto

```bash
npm run dev
```

O projeto será iniciado em:

```txt
http://localhost:5173
```
